# Age Calculator

A precise age calculator built with Next.js. Calculates exact age in years, months, weeks, days, hours, minutes, and seconds.

## Features

- Date of birth input with calendar picker
- Target date input (defaults to today)
- Exact age breakdown: years · months · days
- Total counts: months, weeks, days, hours, minutes, seconds
- Next birthday countdown
- Day of the week you were born

## Deploy to Vercel (3 steps)

1. **Push to GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/age-calculator.git
   git push -u origin main
   ```

2. **Import to Vercel**
   - Go to [vercel.com/new](https://vercel.com/new)
   - Click "Import" on your repository
   - Leave all settings as default — Vercel auto-detects Next.js

3. **Deploy**
   - Click "Deploy" — done in ~30 seconds ✓

## Run Locally

```bash
npm install
npm run dev
# open http://localhost:3000
```
